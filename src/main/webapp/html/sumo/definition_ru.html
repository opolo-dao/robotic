В этом состязании участникам необходимо подготовить автономного робота, способного наиболее эффективно выталкивать робота-противника за пределы белой линии ринга.

В каждом матче соревнуются между собой две команды. Только один участник команды может находиться у ринга, остальные члены команды должны наблюдать из аудитории.
